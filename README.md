# Car Racing Game

A lightweight browser-based **Car Racing Game** built entirely as a **single static HTML file**. The game runs directly in any modern web browser with no installation or dependencies required.

# Live Demo

**Play the game here:**
👉 **https://d3v1k4.github.io/Car-Racing-Game/**


# About the Project

This project demonstrates how an interactive browser game can be created using a single HTML file. It is designed to be simple, fast, and easy to deploy using GitHub Pages.

Whether you're exploring browser game development or looking for a lightweight project, this game showcases the fundamentals of interactive web-based gameplay.


# Features

 Simple and interactive gameplay
 Runs instantly in the browser
 Hosted using GitHub Pages
 Built as a single static HTML file
 No installation required


# How to Play

1. Open the live demo.
2. Start the game.
3. Control the car using the keyboard.
4. Avoid obstacles and survive as long as possible.
5. Try to beat your best score!
